## Notas Importantes para el Cálculo Manual

1. **Autogoles**: Se suman a los goles del equipo contrario
   - Ejemplo: Si el equipo local tiene `home_own_goals = 1`, ese gol se suma a los goles del visitante

2. **Cálculo de Goles Finales**:
   - **Local**: `home_goals + away_own_goals`
   - **Visitante**: `away_goals + home_own_goals`

3. **Determinación del Ganador**:
   - Si `final_local > final_visitante` → Gana el LOCAL
   - Si `final_visitante > final_local` → Gana el VISITANTE
   - Si `final_local = final_visitante` → EMPATE

4. **Puntos**:
   - Victoria = 2 puntos
   - Empate = 1 punto
   - Derrota = 0 puntos

5. **Partido Especial - Match ID 24 (Jornada 3)**:
   - Deportivo Gordon (Local) vs Junior's TJ (Visitante)
   - Marcador: 4-3
   - Autogoles: 1 del visitante (Junior's TJ)
   - **Marcador Final**: 5-3 (el autogol se suma al local)
   - Ganador: Deportivo Gordon
